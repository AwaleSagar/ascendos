# Diagrams

Diagram sources live here as Mermaid (`.mmd`) and are the source of truth.
Rendered exports (SVG/PNG) go in `exports/` and are regenerated rather than
hand-edited. When you change a diagram, change the `.mmd` and update any prose
that describes it (see [CONTRIBUTING](https://github.com/AwaleSagar/ascendos/blob/main/.github/CONTRIBUTING.md)).

| Source | Used on |
|---|---|
| `spine.mmd` | README, architecture index |
| `boot.mmd` | architecture/boot |
| `kernel.mmd` | architecture/microkernel |
| `ipc.mmd` | architecture/ipc |

For the full diagram set, see the master
[blueprint](https://github.com/AwaleSagar/ascendos/blob/main/blueprint/AscendOS-Architecture-Blueprint.md),
which embeds all of them inline.
